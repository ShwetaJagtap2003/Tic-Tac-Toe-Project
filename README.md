🎮 Tic Tac Toe - Ultimate Edition
A modern GUI-based Tic Tac Toe game built with Python and Tkinter featuring smart AI opponent and real-time statistics tracking.

📋 Description
This is a feature-rich Tic Tac Toe game where players can compete against a friend or challenge a strategic AI opponent. The game features an intuitive graphical interface, live score tracking, symbol customization, and smooth gameplay. Built with Python's Tkinter and NumPy for efficient board management.

✨ Key Features
Interactive GUI – Clean, modern interface with color-coded buttons and hover effects
Dual Game Modes – Player vs Player and Player vs CPU with smart AI
Symbol Selection – Choose X or O for Player 1 with dynamic color coding
Live Score Tracking – Real-time updates for Wins, Losses, and Ties
Winning Line Highlight – Visual feedback when a player wins
Smart AI Opponent – Strategic decision engine (Win > Block > Center > Corners)
2-Second Exit Protection – Prevents accidental game closure
Keyboard Shortcuts – R to Restart, F1 for Help, ESC to Exit
Reset Option – Reset scores or start fresh game anytime
Detailed Statistics – Track complete match history

🎯 How to Play
Select game mode (PvP or PvC) and Player 1's symbol (X or O)
Click on any empty square to place your symbol
Get three in a row (horizontal, vertical, or diagonal) to win
Scores update automatically after each game
Use New Game button or press R to play another round
Use Reset Scores to clear all statistics

📊 Game Rules
First player to get three symbols in a row wins
If board fills completely, it's a tie
Player 1 (X) – Blue color
Player 2/CPU (O) – Red color
Best of any number of games – track your win/loss record

🛠️ Built With
Python 3
Tkinter (GUI framework)
NumPy (Board management)
